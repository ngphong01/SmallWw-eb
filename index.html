📋 Accordion Component

HTML5 cung cấp nhiều thẻ semantic giúp cấu trúc trang web rõ ràng hơn:

  • <header>: Phần đầu trang hoặc section
  • <nav>: Menu điều hướng
  • <main>: Nội dung chính
  • <article>: Bài viết độc lập
  • <section>: Phần của trang web
  • <aside>: Thanh bên
  • <footer>: Phần cuối trang

CSS cung cấp nhiều selector mạnh mẽ:

/* Pseudo-classes */
:nth-child(odd) { background: #f0f0f0; }
:hover, :focus { color: blue; }
:not(.exclude) { margin: 10px; }

/* Pseudo-elements */
::before, ::after { content: ""; }
::first-letter { font-size: 2em; }
::selection { background: yellow; }

Nguyên tắc thiết kế responsive:

  1. Mobile First: Thiết kế cho mobile trước
  2. Flexible Grid: Sử dụng % thay vì px
  3. Media Queries: Breakpoints cho các thiết bị
  4. Flexible Images: max-width: 100%
  5. Touch-friendly: Kích thước tối thiểu 44px
