CSS Cơ Bản

CSS định dạng giao diện website.

Selectors

  • Tag: p { color: blue; }
  • ID: #header { background: yellow; }
  • Class: .btn { font-size: 16px; }
  • Descendant: div p { color: green; }
  • Child: div > p { color: red; }
  • Adjacent Sibling: h2 + p { color: blue; }
  • General Sibling: h2 ~ p { color: purple; }
  • Pseudo-class: a:hover { color: red; }
  • Pseudo-element: p::first-letter { font-size: 2em; }

Ví dụ: Chữ nổi bật

.highlight { color: var(--primary-color); }
.example::first-letter { font-weight: bold; }

Box Model

Mọi phần tử là một hộp với content, padding, border, margin.

Box Model
.box-model-example {
  width: 100px;
  padding: 10px;
  border: 2px solid var(--primary-color);
  margin: 10px;
}

Display

  • block: Chiếm toàn bộ chiều rộng.
  • inline: Nằm cùng dòng.
  • inline-block: Nằm cùng dòng, có width/height.
  • flex: Sắp xếp linh hoạt.
  • grid: Bố cục lưới.

Media Queries

Responsive design cho các màn hình khác nhau.

@media (max-width: 768px) {
  body { font-size: 14px; }
}

Mẹo & Thủ Thuật

  • Áp dụng box-sizing: border-box toàn cục.
  • Thiết kế mobile-first với media queries.
  • Sử dụng shorthand: margin: 10px 20px;
  • Tối ưu hiệu ứng với transition và transform. 
* { box-sizing: border-box; }
@media (max-width: 768px) { body { font-size: 14px; } }
div { transition: transform 0.3s ease; }
div:hover { transform: scale(1.05); }
